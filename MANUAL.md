## 1. 시작하기

### 설치

설치 과정이 없습니다. `sticky-memo.html` 파일을 브라우저(Chrome·Edge·Firefox·Safari)에서 열면 끝.

### 권장 환경
- **Chrome / Edge 109+** — 모든 기능 지원
- **Firefox** — Web Contacts API 제외 모두 지원
- **Safari** — BMP 저장은 PNG 폴백, 나머지 정상

### 데스크톱 앱처럼 사용
Chrome/Edge 주소창 우측 "앱 설치" 아이콘 클릭 → 독립 창 + 바탕화면 아이콘

### 온보딩
첫 실행 시 자동으로 나타나는 것들:
1. **역할 선택 모달** — 학생·회사원·주부 등 체크 (나중에 변경 가능)
2. **7단계 투어** — 주요 기능 안내
3. **"알고 계셨나요?" 팁** — 20초간 표시

모두 건너뛸 수 있으며, 도움말 센터(❓)에서 언제든 다시 볼 수 있습니다.

## 4. 역할별 도구

앱은 **14개 역할**을 지원하며, 역할마다 사이드바에 전용 도구 섹션이 나타납니다.

### 역할 변경
- 상단바 **👥 아이콘** → "역할 관리" 버튼
- 또는 `Ctrl+K` → "역할 선택"

### 4.1 초등학생
템플릿: 오늘의 일기 · 숙제 체크리스트
도구: D-Day · 장보기

### 4.2 중학생
템플릿: 영단어장 · 과목별 노트
도구: **시간표** · 시험 계획 · D-Day

### 4.3 고등학생
템플릿: 오답노트 · 모의고사 기록
도구: **시간표** · **시험 계획표** (D-Day·과목별 범위) · D-Day

### 4.4 대학생 ⭐ 가장 많은 도구
템플릿: 과제 트래커 · 조별 과제 노트

**시간표** — 학기별 요일×교시 그리드
- 셀 클릭 → 강의명·강의실·교수 입력
- **에브리타임 붙여넣기 자동 파싱**
- PNG 이미지로 저장 가능

**GPA 계산기**
- 4.5 / 4.3 / 4.0 만점 자동 전환
- **목표 GPA 설정** → 다음 학기 필요 평균 계산

**출결 관리**
- 과목별 출석/지각/결석/공결 원클릭
- F 위험 자동 경고 (결석 1/3 이상)

**조별과제**
- 팀원·할일·마감일 통합
- **팀원별 부담 균형 표시**
- 진행률 프로그레스 바

### 4.5 대학원생
템플릿: 논문 노트 · 실험 노트 · 지도교수 미팅

**논문 리딩 리스트**
- 4단계 상태 (예정/읽는중/완독/핵심)
- **BibTeX 자동 생성** (클립보드 복사)
- 진행률 통계

### 4.6 회사원 (주니어)
템플릿: 회의록 · 오늘 할 일

**프로젝트 파이프라인**
- 4단계 칸반 (제안→진행→리뷰→완료)
- **총 매출·단계별 금액·전환율** 자동 계산

**타임 트래킹**
- 프로젝트별 시작/정지
- **최근 7일 바 차트**

### 4.7 회사원 (시니어/매니저)
템플릿: 1:1 미팅 노트 · 주간 리포트 · OKR

도구: 프로젝트 파이프라인 · 견적서 · 타임트래킹 · D-Day

### 4.8 프리랜서 ⭐ 견적서 강력
템플릿: 견적서 · 청구서

**견적서/청구서 위젯** (전용)
- 회사 정보 영구 저장 (이름/대표/사업자번호/계좌)
- 자동 번호 (INV-2025-001)
- 품목 실시간 합계 + VAT 자동
- 5단계 상태 (초안/발송/승인/청구/입금완료)
- **PDF 출력** (인쇄 대화창 → "PDF로 저장")

### 4.9 가정주부
템플릿: 가족 일정 · 요리 레시피

**가계부** (업그레이드)
- 수입/지출 + 10개 카테고리
- **월별 예산 설정** (카테고리별)
- 예산 대비 소비율 바 (80% 노랑, 100% 빨강)
- **반복 거래** (월세·월급·구독)
- **최근 6개월 추이 라인 차트**
- CSV 내보내기

**장보기 리스트** · **주간 식단** · **레시피북**

### 4.10 노년층 (큰 글씨 우선)
템플릿: 병원 예약 (22pt 큰 글씨)

**약 복용 관리**
- 약별 복용 시간대 설정
- 복용 체크 기록
- **주간 준수율 %** 자동 계산
- 브라우저 알림 (권한 허용 시)

**혈압·혈당·체중 기록**
- 정상 범위 자동 판단 (저혈압/정상/고혈압)
- 평균·최저·최고 통계
- 시계열 라인 차트

### 4.11 요리 애호가
**레시피북**
- 8개 카테고리 (한·양·중·일·분·베이킹·음료·기타)
- 조리 시간 + 난이도 ★1~5
- 재료 태그 + 조리법
- **→장보기** 버튼: 재료 일괄 전송 (중복 제외)
- **⏱ 조리 타이머** (완료 시 브라우저 알림)

### 4.12 예술가/창작자
**작품 포트폴리오**
- 카테고리별 필터 (회화·일러스트·사진·디자인·조각·공예)
- 이미지 업로드 or **"🖌 그림판에서" 바로 그려 추가**
- 판매 상태 토글 + 가격
- 총/판매/매출 통계

### 4.13 여행자
**여행 계획**
- 여행별 D-Day + 예산 바
- 기본 체크리스트 5개 (여권·항공·숙박·보험·환전) + 커스텀
- **지출 추가 → 가계부 자동 기록**
- 📝 여행 일기 메모 자동 생성

### 4.14 독서광
**독서 목록**
- 제목/저자/평점(★1~5)/상태(예정·읽는중·완독)
- 인용구 + 감상 분리
- **연간 목표 대비 진행률**
- **월별 12개월 바 차트**
- → 독서 노트 메모 탭 생성

## 5. AI 기능

API 키 연결 필요 (OpenAI · Gemini · Claude 중 하나). 무료 옵션은 Gemini 추천.

### 5.1 AI 어시스턴트 (`Ctrl+Space`)
- **요약** — 핵심만 3~5문장
- **개선** — 자연스럽게 다듬기
- **이어쓰기** — 문체 유지하며 2~3문단
- **제목 추천** — 5개 제안
- **할 일 추출** — 체크박스로 정리
- **플래시카드 생성** — Q/A 쌍 자동

### 5.2 스마트 AI (`Ctrl+Shift+A`)
컨텍스트 자동 감지:
- 텍스트 선택 → 6가지 액션 버튼 (요약/개선/번역/핵심/할일)
- 이미지 선택 → OCR 5가지 모드 (텍스트/수식/표/문서/명함)
- 기본 → AI 모달

### 5.3 AI Vision OCR
손글씨·스크린샷을 텍스트로:
- **텍스트** 추출
- **수식** → LaTeX
- **표** → Markdown
- **문서** → Markdown 구조화
- **명함** → 구조화 JSON

### 5.4 AI 이미지 생성 (그림판 내)
한국어 프롬프트 → SVG 일러스트 생성 → 새 레이어로 추가

### 5.5 AI 회의 노트 (CLOVA Note 스타일)
- 녹음 + 실시간 받아쓰기(ko-KR)
- AI 화자 분리 / 요약 / 키워드 / 액션 아이템

## 6. 그림판

상단바 🪣 아이콘 또는 `Ctrl+K` → "그림판 열기".

### 도구 (13개)
**브러시**: 연필 · 브러시(필압) · 마커 · 에어브러시 · 지우개
**기능**: 채우기 · **그라디언트** · 스포이드 · **마술봉** · 텍스트 · 선택
**토글**: 대칭 그리기(좌우/상하/사방) · 격자

### 도형 (10종)
직선 · 사각형 · 둥근사각형 · 원/타원 · 삼각형 · 다이아몬드 · 화살표 · 별 · 하트 · 다각형

### 레이어 시스템
- 추가/복제/삭제/가시성
- **16종 블렌드 모드** (Multiply / Screen / Overlay / ...)
- **투명도 슬라이더**
- **병합** (아래와 병합 / 전체 평탄화)

### 필터 (11종)
**기본**: 반전색 · 흑백 · 세피아 · 밝기 · 대비
**고급**: 블러 · 샤프닝 · 노이즈 · 픽셀레이트 · 엠보스 · 윤곽선 감지

### 변형
- 크기 변경 · 자르기 (선택 영역)
- 회전 90°/180°/270° · 좌우/상하 반전
- **자동 배경 제거** (엣지 샘플링)

### 저장
- **저장 = 변환기 자동 오픈** (크기/용량/포맷 조정)
- **빠른 PNG 저장** (`Ctrl+Shift+S`)
- 메모에 직접 삽입 (`Ctrl+Enter`)

### 단축키
`P` 연필 · `B` 브러시 · `E` 지우개 · `G` 채우기 · `I` 스포이드 · `T` 텍스트 · `S` 선택 · `L` 직선 · `W` 마술봉
`Ctrl+Z/Y` Undo/Redo · `Ctrl+C/V` 복사/붙여넣기
`Ctrl+휠` 줌 · `Space+드래그` 팬

## 7. 명함 관리

상단바 🆔 아이콘.

### 입력 방법 (4가지)
1. **파일 업로드**
2. **카메라 촬영** (모바일 후면 카메라 우선)
3. **드래그 앤 드롭**
4. **`Ctrl+V`** 클립보드 이미지

### AI 자동 추출
"AI로 자동 추출" 버튼 → 이름·회사·부서·직책·전화·이메일·주소·LinkedIn 등 12개 필드 자동 채움

### SNS (16개 플랫폼)
LinkedIn · X(Twitter) · Facebook · Instagram · KakaoTalk · WeChat · WhatsApp · Line · Telegram · GitHub · YouTube · TikTok · Naver · Threads · Slack · Discord

URL 붙여넣기 시 자동으로 올바른 플랫폼 칸으로 이동.

### 스마트폰 연락처 연동
**iOS**: vCard(.vcf) 다운로드 → 파일 탭 → 연락처 자동 추가
**Android**: Web Contacts API(Chrome) 또는 vCard
**공통**: QR 코드 → 카메라 스캔 → 연락처 저장
**Web Share**: AirDrop, 카톡, 메시지 직접 공유

### 기타
- 그룹·태그·즐겨찾기
- 미팅 히스토리 (날짜·장소·메모)
- 전화/이메일/지도/LinkedIn 바로가기
- vCard / CSV 가져오기·내보내기
- 내 명함 QR 코드 공유
- 통계 (상위 회사/태그)

## 8. 이미지 변환기

상단바 🔲 아이콘 또는 그림판의 "저장" 버튼.

### 크기 변경
- 너비×높이 직접 입력 + 비율 잠금
- **8개 프리셋**: FHD · HD · 4K · 인스타(1:1, 4:5) · 트위터 · 유튜브 썸네일 · 카카오
- **6개 배율**: 25% ~ 200%

### 목표 용량 (자동 품질 조절)
**50KB / 100KB / 300KB / 500KB / 1MB / 2MB / 5MB**
→ JPEG 품질 이진탐색 + 크기 축소 폴백으로 정확히 맞춤

### 포맷
- **JPG** (용량 조정 가능, 투명 미지원)
- **PNG** (무손실, 투명)
- **WebP** (최신, 용량/품질 균형)
- **BMP** (완벽 호환)

### 입력
파일 업로드 · 드래그앤드롭 · `Ctrl+V`

### 출력
파일 다운로드 · 메모 삽입 · Data URL 클립보드 복사

## 9. 생산성 도구

### 포모도로
25분 집중 + 5분 휴식 타이머. 상단바에 카운트다운 표시.

### 집중 모드 (`Ctrl+/`)
방해 요소 숨김. 현재 탭만 표시.

### 찾아바꾸기 (`Ctrl+F`)
- 정규식 · 대소문자 · 단어 단위
- 한글 단어 경계 지원

### 플래시카드
Q/A 쌍으로 학습. AI로 자동 생성 가능. **메모로 저장**해 재학습.

### 워드 클라우드
자주 쓰는 단어 시각화.

### 번역
한글 ↔ 영어 등 다국어. Google/Papago/DeepL 외부 링크도 제공.

### 목차 (TOC)
제목(H1~H3) 자동 추출. 사이드 패널로 빠른 이동.

### 분할 보기
두 탭을 나란히 표시.

### 갤러리 뷰
탭 썸네일. 7가지 스마트 필터.

## 10. 고급 기능

### 캘린더
- `Ctrl+J` 또는 상단바 📅
- **길게 누르면** 드롭다운: 월간/오늘메모/날짜삽입/캘린더태그
- 메모에 "2025-06-15" 같은 날짜 자동 인식 → 해당 날짜에 표시

### @멘션
메모에 `@`을 입력하면 자동완성 팝업:
- 명함 이름/회사 검색
- 탭 이름 검색
- 삽입 후 클릭으로 해당 명함/탭으로 이동

### 손글씨/펜 모드
- 터치스크린·스타일러스 지원
- **필압 감지** (Pointer Events API)
- **팜 리젝션** (손바닥 터치 무시)
- 자동 감지: 스타일러스 첫 입력 시 토스트 제안

### 300+ 특수문자
수학 · 그리스 · 화살표 · 도형 · 단위

### 15 수식 템플릿
분수 · 적분 · 행렬 · 극한 · 조건식

### 링크 프리뷰
URL 붙여넣기 → YouTube/GitHub 썸네일 카드 자동 생성

### QR 공유
현재 메모를 QR 코드로.

### 인앱 웹 브라우저
8개 검색엔진 (Google · Wikipedia · 나무위키 · GitHub 등) 내장 iframe.

### 탭 암호화 (AES-GCM)
우클릭 → 잠금. 비밀번호로 암호화.

### Windows 폰트 자동 로드
`queryLocalFonts()` API로 시스템 폰트 전체 사용.

## 11. 데이터 관리

### 저장 위치
**브라우저 LocalStorage** (컴퓨터 내부 · 약 5MB 한계)
- 앱을 닫아도 자동 저장
- 브라우저 삭제 시 데이터 손실 주의

### 백업
- **전체 내보내기**: JSON 파일 다운로드
- **전체 가져오기**: 다른 기기에서 복원
- 이미지·오디오 포함

### 탭 단위 내보내기
- PDF / Word(DOCX) / HTML / Markdown / SRT / TXT

### 용량 모니터링
상단바 오른쪽에 현재 저장 크기 표시.
90% 이상 시 휴지통/히스토리 정리 권장.

### 히스토리 / 휴지통
- **버전 히스토리**: 주기적 스냅샷 복원
- **휴지통**: 삭제된 탭 복구

### 자동 저장
- 메모: 입력 후 0.4초 뒤 자동 저장
- 그림판: 30초마다 (복원 제안)

## 12. FAQ

**Q. 여러 기기 간 동기화 되나요?**
A. 브라우저 LocalStorage 기반이라 자동 동기화는 없습니다. JSON 내보내기/가져오기로 수동 이전 가능.

**Q. 오프라인에서 작동하나요?**
A. AI·웹 브라우저·QR API 제외 모두 작동. 단일 HTML 파일이라 인터넷 불필요.

**Q. 데이터를 안전하게 보관하려면?**
A. 민감한 메모는 탭 암호화(AES-GCM) 사용. 전체 JSON 백업을 클라우드 드라이브에 저장 권장.

**Q. 모바일에서도 쓸 수 있나요?**
A. Chrome/Safari로 파일 열기 가능. 홈 화면에 추가하면 앱처럼 사용. 일부 모달은 좁은 화면 자동 대응.

**Q. 기능을 줄이고 싶어요. 너무 복잡해요.**
A. 역할을 선택하지 않으면 기본 메모장만 보입니다. 사이드바의 위젯도 숨길 수 있습니다.

**Q. 기능을 어디서 찾죠?**
A. **`Ctrl+K`** (명령 팔레트) 또는 **❓ 도움말 센터**에서 검색.

**Q. 업데이트는 어떻게?**
A. [GitHub 저장소](https://github.com/auto1225/memo)에서 최신 `sticky-memo.html` 다운로드.

**Q. 버그 신고/기능 제안?**
A. GitHub Issues.
