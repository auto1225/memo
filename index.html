궁금한 점

네, 평생 무료입니다. 현재 광고 없이 운영되고 있고, 추후 사용자가 많아져도 광고는 비침습적인 방식(예: 푸터 후원사 노출)으로만 고려할 예정입니다. 핵심 기능은 평생 무료 보장.
로그인하지 않으면 브라우저 localStorage에만 저장됩니다 (외부 전송 없음). 로그인하면 Supabase (PostgreSQL + Row Level Security)에 본인만 접근 가능한 형태로 저장됩니다. 민감한 탭은 AES-GCM으로 추가 암호화 가능.
OpenAI (GPT-4o, GPT-4 Turbo, GPT-3.5 등), Anthropic (Claude Opus, Sonnet, Haiku), Google (Gemini Pro, Flash 등) 총 16개 모델을 직접 API 키 입력 시 사용 가능합니다. API 키는 본인 브라우저에만 저장되며 우리 서버를 거치지 않습니다.
반대로 가장 모던한 선택입니다. 인터넷 없어도 작동, 다운로드 후 USB로 옮겨 어디서나 사용, 빌드 과정 없음, 의존성 없음, 1개 파일만 백업하면 됨. 850KB 안에 50+ 기능이 들어 있습니다.
데스크톱 Chrome/Edge: 주소창 우측 "설치" 아이콘 클릭. iOS Safari: 공유 버튼 → "홈 화면에 추가". Android Chrome: 메뉴 → "홈 화면에 추가". 일반 앱처럼 작동합니다.
네, GitHub에 공개되어 있습니다 (github.com/auto1225/memo). 직접 다운로드해서 자체 호스팅도 가능합니다.
텍스트는 복사 붙여넣기로 즉시 가능합니다. Markdown · vCard · 이미지 등 표준 포맷 가져오기 지원. 명함의 경우 vCard (.vcf) 파일 가져오면 자동으로 명함 카드로 변환됩니다.
GitHub Issues (github.com/auto1225/memo/issues)에 남겨주세요. 작은 1인 프로젝트라 모든 요청을 다 반영하긴 어렵지만 모두 읽고 우선순위에 반영합니다.
