# JustANotepad — Features Catalog

최종 업데이트: 2026-04-22
버전: **v1.0.33**
총 명령 팔레트 등록 수: **약 180+ 개** (기본 100개 + 이번 확장 65개 + 역할 도구 19개)

> 이 문서는 `app.html` 에 내장된 모든 기능을 카테고리별로 정리합니다.
> 모든 기능은 `Ctrl+K` (macOS: `⌘K`) **명령 팔레트**에서 검색해 바로 실행할 수 있습니다.
> 상세한 시나리오별 사용법은 [`MANUAL.md`](MANUAL.md) 를 참고하세요.

## 찾는 법

**가장 빠른 방법**: `Ctrl+K` 누르고 한글/영어 키워드 한 단어만 입력.

| 키워드 예시 | 나타나는 명령 |
|---|---|
| `저장` | 지금 저장 / PDF / DOCX / HTML / MD / JSON 백업 / 저장 폴더 선택 |
| `그림` | 그림판 열기 / 선택 이미지 편집 / 손글씨 / 스케치 / 이미지 변환기 |
| `AI` | AI 어시스턴트 / AI 스마트 작업 / AI 계정 연결 / AI 이미지 생성 / 명함 AI 스캔 |
| `포스트잇` `JustPin` `postit` | 새 JustPin / 선택을 JustPin으로 / JustPin 전체 숨김 / 자동시작 |
| `테마` | 테마 변경(사이클) / lavender / butter / rose / mint / noir / ocean |
| `배경` | 줄노트 / 모눈종이 / 점격자 / 무지 / 오선지 / 코넬노트 |
| `역할` | 역할 선택 / 역할 대시보드 / [역할] 각 도구 |
| `회의` | 회의 노트 (녹음+STT+AI요약) / 회의록 템플릿 / 주간 리포트 |
| `학점` `gpa` | GPA 계산기 |
| `D-Day` `dday` | D-Day 관리 |

## 명령 팔레트 구조 (개발자용)

`app.html` 안의 `COMMANDS` 배열이 중심 스토어입니다.

```js
// 기본 API
window.justanotepadPalette = {
  open()  // 팔레트 열기
  close() // 팔레트 닫기
  toggle()
  register({ icon, title, description, shortcut, keywords, run })
};
```
