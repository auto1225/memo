그림판 (Paint)

레이어 + 16 블렌드 모드 + 그라디언트 + 필터 11종 + AI 이미지 생성. 포토샵 축소판.

📇

명함 관리

AI로 명함 사진에서 자동 추출. 16개 SNS 플랫폼 지원. vCard/CSV 가져오기·내보내기.

🎤

회의 노트

CLOVA Note 스타일. 녹음+실시간 받아쓰기+AI 화자분리+요약+액션 아이템 자동 추출.

🔲

이미지 변환기

크기·용량·포맷 지정. 목표 용량(500KB 등) 입력하면 JPEG 품질 자동 조절.

🔐

탭 암호화

민감한 메모는 AES-GCM으로 암호화. 비밀번호 없이는 아무도 못 봄.

🌐

인앱 웹 브라우저

8개 검색엔진 내장. 메모에서 나가지 않고 Google·Wikipedia·GitHub 검색.

📲

PWA · 오프라인

홈 화면에 설치 가능. 인터넷 없어도 작동. 새 버전은 자동 알림.
