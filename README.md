# Sticky Memo Ultra

브라우저에서 바로 쓰는 올인원 메모장. HTML 파일 하나로 작동하는 단일 파일 앱.

## 주요 기능

- **리치 텍스트 편집**: 글꼴·크기(pt)·색상·정렬·첨자·제목(H1~H3)
- **배경 6종**: 줄노트·모눈·점격·무지·오선지·코넬 노트
- **손글씨/펜 모드**: 필압 감지, 팜 리젝션, Bézier 부드러운 곡선
- **AI Vision OCR**: 손글씨 → 텍스트/LaTeX 수식/Markdown 표/문서
- **회의 노트 (CLOVA Note 스타일)**: 녹음 + 실시간 받아쓰기 + AI 화자분리/요약/키워드/액션
- **멀티 AI**: ChatGPT / Gemini / Claude 동시 관리
- **인앱 웹 브라우저**: 8개 검색엔진 (Google / Wikipedia / 나무위키 / GitHub 등)
- **탭/워크스페이스/태그/즐겨찾기**
- **15 수식 템플릿**: 분수·적분·행렬·극한·조건식 등
- **300+ 특수문자**: 수학·그리스·화살표·도형·단위 등
- **파일 첨부/화면 캡쳐/스케치/음성 녹음**
- **갤러리 뷰**: 썸네일 카드, 7 스마트 필터
- **내보내기**: PDF / Word(DOCX) / HTML / Markdown / SRT / TXT
- **캘린더**: 날짜 자동 인식, 오늘 메모
- **링크 프리뷰 카드** (YouTube/GitHub 썸네일)
- **찾아바꾸기** (정규식·대소문자·단어 단위)
- **플래시카드·워드 클라우드·포모도로**
- **탭 암호화** (AES-GCM)
- **Windows 전체 폰트 지원** (queryLocalFonts API)
- **이미지 드래그 리사이즈 핸들**

## 사용법

`sticky-memo.html` 파일을 브라우저에서 열면 바로 사용 가능.

### 권장 사용 환경
- Chrome / Edge 109+ (MathML, queryLocalFonts 지원)
- 마이크·스타일러스는 기기/브라우저 권한 필요
- AI 기능은 OpenAI/Gemini/Claude 중 하나의 API 키가 필요 (무료 옵션은 Gemini 추천)

### 바탕화면 스티키 메모로 쓰기
1. 파일을 크롬/엣지로 열기
2. 메뉴 → 저장 및 공유 → "앱으로 설치"
3. 독립 창으로 뜨면서 바탕화면 아이콘 생성

## 파일 구조
