JustANotepad Web Clipper — 개인정보 처리방침

최근 수정일: 2026년 4월 20일

이 문서는 브라우저 확장 "JustANotepad Web Clipper"가 수집·저장·전송하는 데이터에 관해 설명합니다.

1. 수집하는 정보

확장은 사용자가 명시적으로 저장 버튼/단축키/컨텍스트 메뉴를 누를 때에만 다음을 수집합니다:

  • 현재 탭의 URL과 <title>
  • 해당 페이지의 본문 HTML/텍스트 (전체 페이지 모드일 때) 또는 사용자가 선택한 영역
  • 사용자가 입력한 메모 · 태그 (팝업 UI)

페이지를 단순히 방문하거나 저장 버튼을 누르지 않는 한 아무 것도 전송되지 않습니다.

2. 인증 정보

확장은 JustANotepad 웹사이트의 로그인 세션 토큰을 chrome.storage.local에 저장합니다. 이 토큰은 사용자가 직접 justanotepad.com/extension-connect에서 "연결"을 누를 때에만 전송되며, 저장된 클립을 본인 계정에만 귀속시키는 용도로만 사용합니다.

3. 전송 대상

수집된 클립 데이터는 Supabase(사업자: Supabase Inc., 미국)에 호스팅된 JustANotepad의 데이터베이스로만 전송됩니다. 제3자 분석 서비스나 광고 네트워크는 사용되지 않습니다.

4. 보관 기간

저장된 클립은 사용자가 직접 삭제할 때까지 데이터베이스에 남아있습니다. JustANotepad 계정을 삭제하면 연관된 모든 클립도 ON DELETE CASCADE로 함께 삭제됩니다.

5. 권한 설명

  • activeTab + scripting: 저장 버튼을 누른 탭의 본문을 한 번 읽기 위해
  • storage: 로그인 토큰을 로컬에 저장하기 위해
  • contextMenus: 우클릭 메뉴 3종 추가
  • notifications: 저장 성공/실패 알림 표시
  • <all_urls>: 어느 웹 페이지에서든 저장할 수 있게
